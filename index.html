Previsão de Inademplência

A inademplência representa um dos maiores desafios no ambito da gestão de risco bancário, sobre tudo pela dificuldade em antecipar com precisão o comportamento de pagamento dos clientes. Para as instituições financeiras, a inademplência traduz-se em perdas monetárias directas, constragimento de liquidez, em situações extremas, risco de insolvência.

Este projecto tem com finalidade, desenvolver um modelo predictivo de risco de inademplência, identificando os factores mais relevantes associados ao imcumprimento. O estudo foi conduzido com recurso a Regressão Logística e a aprendizagem de máquina supervisionada, enfrentando um cenário forte de desproporção dos dados, onde cerca d 4% dos dados eram os inademplentes.

Depois de ver que mesmo com 93% de precisão na previsão dos pagamentos, isso foi devido a proporção da ademplência e inademplência, corrigindo essa proporção de forma aleatória os resultados mudaram de forma muito notória, com quase 59% de precisão dos inademplentes, o banco usando esse modelo poderá identificar aproximadamente 40 inademplentes a cada 100 existentes. Os resultados do modelo final mostraram 85% de precisão nas previsões, também mostrou que a cada 100 ademplentes, aproximadamente 14 foram previstos como inademplentes.

🏦 Modelo de Previsão de Risco de Inademplência

Visão Geral

Desenvolvimento de modelo de aprendizagem de máquina supervisionada para a previsão de inademplentes. A base de dados muito desbalenceada com aproximadamente 1% dos casos sendo 6% de inademplentes.

Desafio

O modelo inicial apresentou 93% de precisão, mas com apenas 1% de especificidade, um clássico paradoxo de precisão em cenarios de dados desbalanceados.

Solutção

Implementado uma abordagem abrangente, incluindo:

  • • Técnica de balanceamento de dados,
  • • Threshold optimization using ROC analysis
  • • Ensemble methods (Random Forest, XGBoost)
  • • Cost-sensitive learning

Results

PRECISÃO: 0.93 → 0.76
ESPECIFICIDADE: 0.01 → 0.59
KAPPA: 0.02 → 0.28

Tech Stack

R Regressão Logística Aprendizagem de Máquina Supervisionada

BImpacto no negócio

Potencial reduçao de $4.2M em clientes, aproximadamente a 10,000. Melhorou no processo de tomada de decisão em aprovações de créditos.
